# qsort Usage

이 예제는 직접 퀵 정렬을 구현하기보다 C 표준 라이브러리의 `qsort` 함수를 사용해 정수를 정렬합니다.

## How It Works

1. 정렬할 배열과 원소 개수를 준비합니다.
2. 두 원소의 대소 관계를 판별하는 비교 함수를 정의합니다.
3. `qsort`에 배열 시작 주소, 원소 개수, 원소 크기, 비교 함수를 전달합니다.
4. 라이브러리가 내부 정렬 알고리즘을 사용해 배열을 정렬합니다.

## Characteristics

- 표준 라이브러리를 사용하므로 재사용이 쉽습니다.
- 다양한 자료형을 같은 인터페이스로 정렬할 수 있습니다.
- 내부 구현은 플랫폼마다 다를 수 있으며, 반드시 퀵 정렬이라고 단정할 수는 없습니다.
- 사용자는 비교 함수의 정확성을 보장해야 합니다.
