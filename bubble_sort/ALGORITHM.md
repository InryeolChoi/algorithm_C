# Bubble Sort

버블 정렬은 인접한 두 원소를 비교해서 순서가 잘못되어 있으면 서로 교환하는 방식의 정렬입니다.

## How It Works

1. 배열의 처음부터 끝까지 인접한 두 값을 비교합니다.
2. 앞의 값이 더 크면 두 값을 교환합니다.
3. 한 번 끝까지 순회하면 가장 큰 값이 맨 뒤로 이동합니다.
4. 마지막에 정렬이 완료된 구간을 제외하고 같은 작업을 반복합니다.

## Characteristics

- 구현이 단순합니다.
- 평균 및 최악 시간 복잡도는 `O(n^2)`입니다.
- 추가 메모리가 거의 필요 없는 제자리 정렬입니다.
- 데이터가 많아질수록 비효율적입니다.
