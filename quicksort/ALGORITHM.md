# Quick Sort

퀵 정렬은 하나의 피벗을 기준으로 작은 값과 큰 값을 나누고, 나뉜 구간을 재귀적으로 정렬하는 분할 정복 방식의 정렬입니다.

## How It Works

1. 배열에서 피벗을 하나 선택합니다.
2. 피벗보다 작은 값은 왼쪽, 큰 값은 오른쪽으로 분할합니다.
3. 피벗이 최종 위치에 놓이도록 배열을 재배치합니다.
4. 피벗을 기준으로 나뉜 왼쪽과 오른쪽 구간에 대해 같은 작업을 반복합니다.

## Characteristics

- 평균 시간 복잡도는 `O(n log n)`입니다.
- 최악 시간 복잡도는 `O(n^2)`입니다.
- 추가 배열 없이 구현할 수 있는 제자리 정렬입니다.
- 실제로는 매우 빠른 편이라 많이 사용됩니다.
