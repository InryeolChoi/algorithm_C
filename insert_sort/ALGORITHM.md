# Insertion Sort

삽입 정렬은 앞쪽의 정렬된 구간에 현재 원소를 적절한 위치로 끼워 넣는 방식의 정렬입니다.

## How It Works

1. 두 번째 원소부터 시작합니다.
2. 현재 값을 임시로 저장합니다.
3. 앞쪽의 정렬된 구간을 뒤에서부터 확인하며, 현재 값보다 큰 원소를 한 칸씩 뒤로 밀어냅니다.
4. 현재 값이 들어갈 위치를 찾으면 그 자리에 삽입합니다.
5. 배열 끝까지 반복합니다.

## Characteristics

- 거의 정렬된 데이터에서는 효율이 좋은 편입니다.
- 평균 및 최악 시간 복잡도는 `O(n^2)`입니다.
- 추가 메모리가 거의 필요 없는 제자리 정렬입니다.
- 작은 입력 크기에서 자주 사용됩니다.
