# Quick Sort

이 예제는 마지막 원소를 피벗으로 선택하고, 피벗보다 작은 값을 앞쪽으로 모으는 방식의 퀵 정렬입니다.

## How It Works

1. 구간의 마지막 원소를 피벗으로 선택합니다.
2. 왼쪽부터 순회하며 피벗보다 작은 값을 앞쪽으로 이동시킵니다.
3. 순회가 끝나면 피벗을 경계 위치와 교환합니다.
4. 피벗을 기준으로 나뉜 두 구간을 재귀적으로 정렬합니다.

## Characteristics

- 평균 시간 복잡도는 `O(n log n)`입니다.
- 최악 시간 복잡도는 `O(n^2)`입니다.
- 파티션 구현 방식에 따라 실제 성능과 교환 횟수가 달라집니다.
- 재귀 호출을 사용하므로 입력 크기에 따라 호출 깊이가 커질 수 있습니다.
